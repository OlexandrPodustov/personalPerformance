# Security Policy

This repository contains personal practice code (coding exercises, katas, and
self-study projects) and is not deployed as a production service.

## Reporting a Vulnerability

If you find a security issue in this repository (e.g. a vulnerable dependency
or unsafe code pattern), please report it privately via GitHub's
[private vulnerability reporting](https://github.com/OlexandrPodustov/personalPerformance/security/advisories/new)
instead of opening a public issue.

I'll do my best to respond within a few days, but there's no formal SLA since
this is a personal, non-commercial repository.
